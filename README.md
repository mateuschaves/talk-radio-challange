# Talk Radio Challenge
Teste prático para a vaga de Desenvolvedor Backend NodeJS Pleno


# Instalação

```bash
 $ git clone https://github.com/mateuschaves/talk-radio-challenge.git
```

```bash
$ cd talk-radio-challenge
```

```bash
# yarn
$ yarn install

# npm
$ npm install
```

# Execução

```bash
# yarn
$ yarn start:dev

# npm
$ npm run start:dev
```

# Compilação

```bash
# yarn
$ yarn build

# npm
$ npm run build
```

# Testes

```bash

# yarn

$ yarn test

# npm

$ npm run test
```
